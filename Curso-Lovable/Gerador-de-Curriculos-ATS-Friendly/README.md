# Gerador de Currículos ATS Friendly com Lovable

Projeto desenvolvido como parte de um desafio da DIO, utilizando o Lovable para criar uma aplicação web simples, moderna e responsiva para geração de currículos profissionais.

## Sobre o projeto

O **CurrículoFácil ATS** permite que o usuário preencha suas informações pessoais, experiências profissionais, formação acadêmica, habilidades e idiomas para gerar um currículo organizado e adequado para sistemas de recrutamento ATS.

A aplicação não exige cadastro e não armazena permanentemente os dados informados.

## Funcionalidades

- Preenchimento de informações pessoais;
- Inclusão de objetivo profissional;
- Cadastro de experiências profissionais;
- Cadastro de formação acadêmica;
- Inclusão de habilidades;
- Inclusão de idiomas;
- Pré-visualização do currículo;
- Limpeza dos campos preenchidos;
- Impressão ou salvamento do currículo em PDF;
- Interface responsiva para computadores e celulares.

## Objetivo do desafio

O objetivo deste projeto foi utilizar prompts claros e estruturados para orientar uma ferramenta de desenvolvimento com inteligência artificial na criação de uma aplicação web funcional.

## Observações

Este é um projeto demonstrativo desenvolvido para fins educacionais. Os dados preenchidos pelo usuário permanecem apenas durante a utilização da aplicação e não são armazenados em banco de dados.
